A from scratch community implementation of the now-discontinued NeoForge Fluid API. Should allow 1.19+
mods to port to 26.1+ versions of Minecraft with minimal changes. See the [example mod][em] for help.

This is a community driven project, and contributors are welcome.

Check the [wiki][wiki] for documentation!
